# 서비스 기획서

## 1. 서비스 슬로건
> "안전한 만남 100% 보장"

## 2. 모드 구조
- **친구모드**:
    - 무료 가입 가능
    - 친구 찾기 중심, 보증금 없음
- **맞선모드**:
    - 보증금 필수
    - 안전 보장된 매칭 & 오프라인 만남 지원
    - 정원 제한 (예: 남자 50명, 여자 60명)

## 3. 보증금 정책
- 맞선모드 가입 시 보증금 예치 (예: 15,000원)
- 신고 1회당 5,000원 차감
- 3회 누적 시 강제 탈퇴 & 보증금 몰수
- 정상 종료 시 환급 가능
- 6개월 이상 비활성화 시 보증금 몰수

## 4. 신고 및 제재
- 신고 시 증거 자료 첨부(스크린샷 등)
- 운영자 검토 후 제재 확정 → 보증금 차감

## 5. 오프라인 만남
- 제휴 카페 한정 운영 (룸 형태, 프라이버시 보장)
- 초기: 서울 1곳 제휴 → 이후 확장 가능
- 첫 만남: 운영자/스태프 포함 3자 만남
- 이후 만남: 위치 공유 + 긴급 호출 기능 지원

## 6. 뱃지 정책
- 커플 성사 시 자동 뱃지 발급
- 뱃지 공개는 의무 (숨김 불가)
- 해제는 상호 합의 또는 운영자 승인 시만 가능
