# 화면 정의서 (Wireframe & Flow)

## 1. 회원가입 / 로그인
- **입력 필드**: 이메일, 비밀번호, 비밀번호 확인
- **버튼**: 회원가입, 로그인
- **안내 문구**: "친구모드 먼저 체험할 수 있어요"
- **에러 처리**:
    - 이메일 형식 오류
    - 비밀번호 불일치

---

## 2. 홈 (친구모드)
- **상단바**: 로고, 알림, 프로필
- **탭**: 친구모드 / 맞선모드
- **카드 구성**: 프로필 사진, 닉네임, 자기소개, [친구 요청 버튼]
- **하단바**: 홈, 검색, 채팅, 커뮤니티, 내 정보

---

## 3. 맞선모드 신청
- **화면 요소**:
    - 보증금 안내 박스
    - 결제 버튼
    - 환불 정책 안내
- **동작**:
    - 신청 인원이 정원 초과 → 대기 순번 안내
    - 성공 시 맞선모드 전환

---

## 4. 프로필 작성
- **입력 요소**: 닉네임, 성별, MBTI, 혈액형, 자기소개(복수), 사진 업로드
- **버튼**: 저장
- **제약**: 닉네임 중복 불가, 자기소개 최소 1개 필수

---

## 5. 커뮤니티 / 뱃지 노출
- **리스트**: 글 카드 (닉네임 + 뱃지 + 글 미리보기)
- **작성**: 글쓰기 버튼 (FAB)
- **정책**: 뱃지는 항상 노출, 숨김 불가

---

## 6. 매칭 결과 / 오프라인 약속
- **화면 요소**: 상대 프로필 카드, 채팅 시작, 제휴 카페 예약 안내
- **안전 기능**: 위치 공유, 긴급 호출 버튼
