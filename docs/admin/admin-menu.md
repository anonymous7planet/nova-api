# 관리자 메뉴 구조 (Admin Menu)

관리자 시스템은 **4단계 권한 체계(최고 관리자 / 관리자 / 매니저 / 운영자)**를 기반으로 운영됩니다.  
각 메뉴는 권한에 따라 접근 가능 여부가 달라집니다.

---

## 메뉴 트리 구조

- 관리자 대시보드
    - 회원 관리
        - 전체 회원 목록
            - 친구 모드 회원
            - 맞선 모드 회원
        - 가입 대기 회원 목록
            - 친구 모드 가입 대기
            - 맞선 모드 가입 대기
        - 탈퇴 회원 목록
        - 추방 회원 목록
        - 맞선 완료 회원 목록
        - 회원 상세 보기 / 수정 / 상태 변경
    - 매칭 관리
        - 자동 매칭 현황
        - 수동 매칭 (최고 관리자 전용)
        - 매칭 문항 관리
            - 문항 등록
            - 문항 수정
            - 문항 카테고리 관리
        - 매칭 통계
    - 포인트 관리
        - 포인트 정책 관리 (환율, 차감 규칙 등)
        - 포인트 충전 내역
        - 포인트 사용 내역
        - 정산 관리
    - 콘텐츠 관리
        - 배너 관리
        - 이벤트 관리
        - 공지사항 관리
    - 시스템 관리
        - 관리자 계정 관리
            - 계정 등록
            - 권한 설정
            - 활동 로그
        - 앱 버전 관리
        - 로그 관리
        - 환경 설정
    - 통계 / 리포트
        - 회원 통계
        - 매칭 통계
        - 포인트 통계
        - 서비스 이용 패턴 분석
