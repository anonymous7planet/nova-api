# Admin - Matching Management (상세)

## 목적
- 자동매칭 파라미터 관리와 수동(운영자) 매칭 기능 규격

## 1. 자동매칭 설정
- 항목:
    - 기본 필터: 성별, 연령 범위, 활동 지역
    - 가중치 설정: MBTI(0~30), 설문(0~50), 취미(0~20), 기타(0~10) — 총합 정규화
    - 추천 커트라인: 추천 노출 최소 점수(예: 60점)
- UI:
    - 슬라이더/숫자 입력으로 가중치 조정
    - 저장 시 가중치 변경 로그 기록

## 2. 매칭 스케줄
- 자동 스케줄: 매칭 재계산 주기(예: 실시간/매 1시간/일 1회)
- 수동 트리거: 운영자가 특정 회원에 대해 재계산 요청 가능

## 3. 수동 매칭 (SuperAdmin 전용)
- 워크플로우:
    1. SuperAdmin이 회원(A) 조회
    2. 후보 리스트(자동/수동 검색)를 확인
    3. 후보 선택(복수 선택 가능) → Confirm Match
    4. 시스템이 두 유저에게 매칭 제안(운영정책에 따라 자동 확정 혹은 양측 수락 필요 설정)
    5. 매칭 확정 시: 상태 변경(`맞선 완료` 또는 `맞선 활성`), 알림, 예약 옵션 활성화
- 권한/감사:
    - 모든 수동 매칭 작업은 audit log에 저장 (who, when, why)
    - SuperAdmin 외 권한 차단

## 4. 매칭 결과 모니터링
- 지표:
    - 매칭 요청 수, 수락률, 오프라인 만남 예약률, 신고율(매칭 후)
- 리포트:
    - 일/주/월 별 매칭 성사 분석
    - 가중치 변경 후 AB테스트 결과 비교
