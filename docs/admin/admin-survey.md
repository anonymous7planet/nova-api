# Admin - Survey & Question Management

## 목적
- 설문 문항을 운영자가 직접 생성/편집/관리하여 매칭 품질을 운영적으로 개선

## 기능 목록
1. 질문 목록 (QID, 제목, 유형, 상태, 생성일)
2. 질문 생성/편집 화면
    - 유형: 객관식(단일/다중), 5점 척도, 서술형
    - 보기 추가/삭제 및 보기별 ID 부여
    - 보기별 호환도 매트릭스 설정(그리드 에디터)
    - 공개 여부 / 배포 기간 설정
3. 매칭 호환도 매트릭스
    - 각 보기 콤비네이션에 대해 점수 입력
    - 점수 범위: -10 ~ +50 (운영자가 정책에 따라 설정)
    - 입력 편의: CSV 업로드/다운로드 지원
4. 버전 관리
    - 설문 버전별로 응답 이력 보관
    - 과거 버전으로 롤백 가능
5. 통계 & 분석
    - 문항별 응답 분포
    - 문항 상관관계(어떤 문항 조합이 매칭 성공률과 연관이 높은지)
    - 응답율 리포트 (회원군별: 연령/지역/가입일)

## UI 예시: 문항 편집 화면
- 좌측: 문항 텍스트 + 보기 리스트 (편집)
- 우측: 매트릭스 에디터 (보기 × 보기 표)
- 하단: 저장 / 미리보기(사용자 화면에서 보이는 형태)
