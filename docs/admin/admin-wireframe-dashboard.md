# 관리자 대시보드 와이어프레임 설계 (v1)

## 1. 화면 구조 개요

```plaintext
[관리자 대시보드]
 ├─ 상단 헤더
 │   ├─ 로고 / 서비스명
 │   ├─ 관리자명 표시 및 로그아웃 버튼
 │   └─ 알림(Notification) 아이콘
 │
 ├─ 좌측 사이드바
 │   ├─ 대시보드 (현재 페이지)
 │   ├─ 회원 관리
 │   ├─ 매칭 관리
 │   ├─ 로그 관리
 │   ├─ 통계 관리
 │   └─ 시스템 설정
 │
 └─ 메인 패널
     ├─ [요약 카드 영역]
     │    ├─ 신규 가입자 수 (오늘 / 전일 대비 증감률)
     │    ├─ 활성 회원 수
     │    ├─ 매칭 성공률
     │    ├─ 제재 회원 수
     │    └─ 앱 접속 수 (일간)
     │
     ├─ [통계 그래프 영역]
     │    ├─ 일자별 가입 추이 → Line Chart
     │    ├─ 성비 / 연령별 분포 → Pie Chart
     │    ├─ 지역별 분포 → Bar Chart
     │    └─ 국가별 언어 선택 비율 → Donut Chart
     │
     ├─ [실시간 로그 영역]
     │    ├─ 최근 관리자 활동 (등록/수정/삭제)
     │    ├─ 최근 신고 / 제재 이력
     │    └─ 최근 매칭 결과
     │
     └─ [주요 알림 영역]
          ├─ 신고 누적 회원 알림
          ├─ 매칭 실패율 경고
          ├─ 시스템 점검 공지
          └─ 업데이트 예정 기능 안내
```

## 2. 주요 기능 상세 설명

### 2.1 요약 카드
| 카드명 | 지표 설명 | 시각화 방식 | 갱신 주기 |
|--------|------------|-------------|------------|
| 신규 가입자 수 | 오늘 신규 가입자 수 (전일 대비 %) | 숫자 + 화살표 | 1시간 |
| 활성 회원 수 | 최근 7일 로그인 기준 | 숫자 | 1시간 |
| 매칭 성공률 | 전체 매칭 중 성공 비율 | 숫자 + Progress bar | 1시간 |
| 제재 회원 수 | 정지/추방 회원 합계 | 숫자 | 1시간 |
| 앱 접속 수 | 일간 평균 접속자 수 | 숫자 + Line sparkline | 1시간 |

---

### 2.2 통계 그래프
| 그래프명 | 설명 | 형태 | 데이터 출처 |
|-----------|------|------|--------------|
| 일자별 가입 추이 | 최근 30일 가입자 수 | Line Chart | user_table |
| 성비 / 연령별 분포 | 성별 + 나이대별 비율 | Pie Chart | user_profile |
| 지역별 분포 | 시/도별 회원 수 | Bar Chart | user_location |
| 언어/국가 비율 | 언어별 설정 수 | Donut Chart | user_setting |

---

### 2.3 실시간 로그
| 구분 | 표시 내용 | 예시 |
|------|------------|------|
| 관리자 활동 로그 | 등록/수정/삭제 내역 | `매칭 문항 수정 - AdminA (2025-10-10 14:03)` |
| 신고/제재 로그 | 신고 발생 또는 해제 기록 | `UserB 제재 해제 - AdminC` |
| 매칭 로그 | 신규 매칭 또는 실패 | `UserX ↔ UserY 매칭 성공 (93%)` |

---

### 2.4 주요 알림
| 구분 | 설명 |
|------|------|
| 신고 누적 | 특정 회원 신고 횟수 초과 시 알림 |
| 매칭 실패율 | 특정 기간 내 실패율 >80% 시 표시 |
| 시스템 공지 | 업데이트 / 점검 시간 |
| 기능 업데이트 | 새 기능 추가 시 내부 공지 |

---

## 3. 접근 권한
| 관리자 등급 | 접근 가능 여부 |
|---------------|----------------|
| SA (최고 관리자) | 전체 접근 가능 |
| MM (매칭 관리자) | 매칭 관련 데이터 접근 가능 |
| CM (콘텐츠 관리자) | 통계 / 배너 / 공지 접근 가능 |
| SM (고객지원 관리자) | 로그 / 신고 내역 접근 가능 |

---

## 4. 차트 시각화 라이브러리 제안
- **Recharts (React 기반)**
- **Chart.js**

## 5. 반응형 정책
| 디바이스 | 지원 여부 | UI 정책 |
|-----------|-------------|----------|
| PC (웹) | ✅ | 전체 메뉴 지원 |
| 모바일 (P+M 일부) | ⚙️ | 요약 카드 + 통계 그래프만 표시 |
| 태블릿 | ✅ | 축소형 대시보드 자동 구성 |

---

## 6. 추가 고려사항
- 국가별 회원 분포를 나중에 “해외 확장” 데이터로도 사용할 수 있음
- 관리자 페이지 진입 시 로그인 후 세션 기반 권한 체크 필요
- 모든 주요 데이터는 캐시(예: Redis)로 1시간 단위 갱신
