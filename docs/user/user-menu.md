# User Menu (사용자 메뉴 트리 & 설명)

> 목적: 사용자 관점에서 앱 메뉴/하위기능을 계층적으로 정리.  
> 각 항목 아래에는 화면명, 주요 동작, 접근권한, 비고를 표기.

## 1. 진입(비로그인)
- /launch
    - 언어 선택 (한국어 / English / 日本語)
    - 소개(슬로건) 배너
    - 로그인 / 회원가입 버튼
    - (푸시 권한 요청 팝업 - 나중에)

## 2. 인증/회원가입
- /auth
    - 로그인 (이메일 / 비밀번호, OAuth 추후 가능)
    - 회원가입 시작 (친구모드 기본 가입)
        - Step 1: 계정생성 (이메일, PW, 닉네임)
        - Step 2: 본인인증 (내국인: 휴대폰 PASS / 외국인: 외국인등록증+휴대폰)
        - Step 3: 사진 업로드 (최소 3장, 최대 5장) — 얼굴 정면 1장 필수
        - Step 4: 프로필 입력 (분할 페이지, 필수 항목 우선)
        - 가입 완료 → 상태: `가입대기` (운영자 승인 필요)
    - 비밀번호 찾기

## 3. 메인(로그인 후)
- /home
    - 상단바: 로고, 검색, 알림
    - 모드 탭: [친구모드(기본)] — (맞선 모드는 별도 신청)
    - 추천 피드(카드형) — 프로필 카드, 관심사 태그, 근거리/활동지역
    - 오늘의 설문(맞선 회원 전용, 조건 충족 시 팝업)
    - 퀵 액션: 채팅, 예약(맞선용 장소)

## 4. 탐색 / 검색
- /search
    - 필터: 성별, 연령대, 활동지역(3곳), 언어, 취미 태그, MBTI 등
    - 정렬: 매칭확률순 / 최근접속 / 신규회원

## 5. 프로필 & 상호작용
- 프로필 카드 (요약)
    - 행동: 프로필보기, 친구신청(친구모드), 매칭신청(맞선 대상자에게만)
- 프로필 상세
    - 사진 갤러리, 상세 정보(학력/직업/언어/활동지역), 뱃지(커플 등)
    - 행동: 친구요청 / 매칭요청 / 채팅 요청 / 신고

## 6. 채팅
- /chat
    - 매칭된 상대와만 채팅(맞선모드)
    - 친구모드에서는 친구 요청 수락 시 채팅 가능
    - 기능: 텍스트, 이미지, 위치공유(선택), 긴급신고 버튼

## 7. 커뮤니티 (Lounge)
- /community
    - 게시판(글/댓글), 이벤트, 후기(매칭 후기)
    - 노출: 프로필 뱃지 표시 (커플 뱃지 항상 노출)

## 8. 예약 / 오프라인 만남
- /reservation
    - 제휴 카페 목록 (서울 중심, 룸형 우선)
    - 예약 신청 → QR 체크인 발급
    - 운영자 동석 옵션(유료/무료 여부 설정)

## 9. 마이페이지
- /mypage
    - 내 프로필 관리(사진/소개/관심사/언어)
    - 이상형(간단) 설정 (친구모드 기본: 나이/지역)
    - 포인트/결제 내역
    - 알림 설정 / 개인정보 설정
    - 상태 전환: 친구모드 활성 / 맞선 가입신청(추가) / 탈퇴 요청

## 10. 도움말 / 신고
- /help
    - 신고하기 (비속어/스팸/성희롱 등) — 증거(스크린샷) 첨부 필수
    - 자주 묻는 질문(FAQ)
    - 고객센터 문의 (티켓 발급)
