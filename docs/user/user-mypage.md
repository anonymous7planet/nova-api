# User MyPage (마이페이지 상세 규격)

## 목적
- 사용자가 자신의 정보를 관리하고, 맞선 상태/포인트/알림 등을 확인하는 개인 대시보드

## 공통 섹션
- 프로필 요약 (사진, 닉네임, 상태뱃지)
- 포인트 잔액 / 충전 이동 버튼
- 알림 설정 / 푸시 설정
- 계정 설정 (비밀번호 변경, 2FA(선택), 개인정보 처리방침)

## 친구모드 회원 마이페이지
- 프로필 편집 (사진, 자기소개, 활동지역, 관심사)
- 간단 이상형(수정 가능: 나이범위,지역)
- 활동 로그: 받은 친구요청, 보낸 요청, 채팅 기록 요약
- 내 게시글(커뮤니티) 관리

## 맞선모드 회원 마이페이지
- 상세 프로필 편집 (직업, 학력, 생활습관 등)
- 상세 이상형 편집 가능 (나이/직업/학력/종교/키 등)
- 맞선 상태(가입대기 / 활성 / 완료) 및 진행로그
- 포인트 결제/내역 (포인트 충전, 매칭 차감 내역)
- 설문 이력(오늘 응답 여부, 지난 응답 보기(요약))
- 맞선 탈퇴 요청 (운영자 승인 프로세스 필요)

## 개인정보/보안 관련
- 탈퇴 절차: 탈퇴 요청 → 관리자 검토(보증금 반환 등) → 탈퇴 완료
- 보증금 환불 정책 페이지 링크(명확히 표기)
