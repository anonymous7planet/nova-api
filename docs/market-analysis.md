# 시장 경쟁 분석 보고서

## 1. 시장 현황
- 한국 및 글로벌 소개팅/친구만들기 앱 시장 동향
- 외국인 거주자 증가 → 다국어(한국어/영어/일본어) 지원 필요성

## 2. 경쟁 서비스 분석
| 서비스 | 특징 | 한계 |
|--------|------|------|
| Tinder | 글로벌 유저풀, 스와이프 방식 | 안전성 보장 부족, 외국인 유입 많음 |
| Bumble | 여성 우선 매칭 | 국내 활성화 낮음 |
| Noondate(정오의 데이트) | 한국 전통 소개팅 앱 | 안전 보장 부족, 외국인 접근 어려움 |

## 3. 페르소나
- **Persona A (외국인)**: 한국어 배우고 싶음, 친구 사귀기 원함
- **Persona B (한국인 여성)**: 안전이 최우선, 진심 있는 만남 선호
- **Persona C (한국인 남성)**: 신뢰성 있는 매칭, 오프라인 만남 희망

## 4. 차별화 포인트
- 안전 보장 (보증금 제도, 신고 시스템, 뱃지 공개)
- 오프라인 맞선(제휴 카페)
- 커플 인증 뱃지 및 커뮤니티 기능
